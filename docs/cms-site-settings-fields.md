# CMS `siteSettings` fields (portfolio)

From `GET {CMS_PUBLIC_URL}/api/public/site` → `siteSettings`.

| Field | Use |
|-------|-----|
| `projectsSectionTitle` | Home projects block heading |
| `projectsSectionDescription` | Home projects block subtitle |
| `allProjectsPageTitle` | `/projects` page title |
| `allProjectsPageDescription` | `/projects` page intro (optional) |
